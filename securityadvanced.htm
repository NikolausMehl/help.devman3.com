UNREGISTERED EVALUATION VERSION

If a user account has been inactive for three or more months:

•The account will automatically be disabled.

•When trying to log on, users will be notified that their account is inactive.

 

 

The system administrator can reactivate the account as follows:

•Go to Administration > Security > Users.

•Select the user profsle,  and then select the Reactivate option on the menu.

 

 

Please note: This feature will be bypassed for users logging on with LDAP or Azure AD.
